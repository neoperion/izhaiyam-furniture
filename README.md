# IZHAYAM HANDLOOM FURNITURE

## Project info

E-commerce website for premium handcrafted furniture featuring traditional artistry and contemporary design.

## Getting Started

Requirements: Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd izhaiyam-furniture

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build for Production

```sh
npm run build
```

## Features

- Modern e-commerce interface
- Product catalog with categories (Cots, Sofas, Diwans, Chairs, Balcony Furniture, Swings)
- Responsive design
- Contact and about pages
- Newsletter subscription
